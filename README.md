# 《白夜行》数字人文可视化网站

## 项目简介

这是一个基于东野圭吾小说《白夜行》的数字人文可视化项目，通过多种数据可视化技术，将文学作品的内容进行多维度分析和展示，帮助读者更深入地理解这部经典推理小说。

## 功能特色

### 1. 作品概览
- 基本信息展示：作者、译者、出版信息等
- 文本统计数据：字数、章节、人物数量等
- 故事背景介绍：时间、地点、核心主题

### 2. 人物关系网络
- 交互式人物关系图
- 主要人物介绍卡片
- 人物关系强度可视化
- 支持拖拽和缩放操作

### 3. 故事时间线
- 19年故事时间跨度展示
- 关键事件节点标注
- 响应式时间线设计
- 重要情节梳理

### 4. 情感分析
- 雷达图展示情感维度
- 情感强度可视化条形图
- 八大情感维度分析：压抑、黑暗、救赎、希望、悬疑、爱情、悲剧、成长

### 5. 地点分析
- 故事主要地点介绍
- 地点象征意义分析
- 空间与情节关系展示

### 6. 主题词云
- 动态词云展示
- 核心主题标签
- 关键词权重可视化

## 技术实现

### 前端技术栈
- **HTML5**: 语义化页面结构
- **CSS3**: 响应式设计、动画效果
- **JavaScript**: 交互逻辑、数据处理
- **Chart.js**: 情感分析雷达图
- **ECharts**: 人物关系网络图、词云图

### 设计特点
- **响应式设计**: 适配不同屏幕尺寸
- **现代化UI**: 渐变色彩、卡片式布局
- **流畅动画**: 滚动动画、悬停效果
- **交互体验**: 平滑滚动、动态图表

## 数字人文视角

### 文本分析维度
1. **叙事结构**: 时间线分析、情节发展
2. **人物关系**: 社会网络分析、角色重要性
3. **情感特征**: 情感维度量化、情感变化趋势
4. **空间地理**: 故事地点分布、空间象征
5. **主题内容**: 关键词提取、主题聚类

### 学术价值
- 为文学研究提供数据支撑
- 展示数字人文研究方法
- 促进跨学科交流合作
- 增强文学作品的可读性

## 使用说明

### 本地运行
1. 将所有文件放在同一目录下
2. 用浏览器打开 `index.html` 文件
3. 确保网络连接正常（需要加载CDN资源）

### 功能操作
- **导航菜单**: 点击可快速跳转到对应章节
- **人物关系图**: 支持拖拽移动、滚轮缩放
- **时间线**: 展示故事发展的关键节点
- **图表交互**: 鼠标悬停显示详细信息
